# Eventify - פלטפורמת אירועים יוקרתיים 🌟

מערכת Web מקיפה שנועדה להפוך את תהליך הפקת האירועים לפשוט, מאורגן ומקצועי. המערכת משמשת כזירת מפגש דיגיטלית בין בעלי עסקים המציעים שירותי יוקרה לבין בעלי אירועים המחפשים פתרון איכותי תחת קורת גג אחת.

---

## ✨ תכונות עיקריות

### 🤵 עבור בעלי אירועים
* **חיפוש וסינון:** איתור ספקים לפי קטגוריות והתאמה אישית.
* **ניהול יומן:** מעקב אישי אחר תכנון האירוע.
* **מועדפים:** שמירת שירותים נבחרים בלחיצת כפתור.

### 💼 עבור אנשי עסקים
* **ניהול תוכן:** פרסום וניהול דף העסק בצורה עצמאית.
* **עדכון בזמן אמת:** שליטה מלאה בפרטים המוצגים ללקוח.
* **חשיפה:** הגעה לקהל יעד ממוקד המחפש שירותי יוקרה.

### 🔒 אבטחה
* מערכת אימות משתמשים (Login/Register) מאובטחת.
* הצפנת נתונים ושמירה על פרטיות המשתמשים.

---

## 🛠 טכנולוגיות

| צד לקוח (Frontend) | צד שרת (Backend) | בסיס נתונים ואבטחה |
| :--- | :--- | :--- |
| HTML5 & CSS3 | Node.js | MongoDB & Mongoose |
| JavaScript (ES6+) | Express.js | JWT & bcrypt |

## 🚀 איך מפעילים את הפרויקט?

1. בצעו Clone לפרויקט:
`git clone https://github.com/shanimishaely/eventify-event-planning-
platform.git
`

2. התקינו את התלויות:
`npm install`

3. הריצו את השרת:
`npm start`
